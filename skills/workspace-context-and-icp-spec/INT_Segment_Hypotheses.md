# INT — Segment Hypotheses
*Stage 2. First-pass segment guesses. Validated in Stage 3.*

---

## Source priority by tier

**Tier 0 (baseline, where most hypotheses come from):**
- Customer logos page — count, categorize by industry/size/geo.
- Public information about each logo'd customer.
- Press releases naming the client and a customer together.
- Case studies on customer-side websites.
- G2 / Capterra reviews — reviewer's company size / industry.
- Hiring patterns at lookalike companies (signals interest in the category).

**Tier 1:** founder LinkedIn posts mentioning specific customers / wins / use cases.

**Tier 2:** internal case studies, CRM segmentation, sales-team take on best/worst-fit segments.

Tier 0 alone reaches solid hypotheses — Tier 1/2 mainly validates them.
